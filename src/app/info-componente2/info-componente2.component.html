Informacion del proyecto

Las imagenes proporcionadas corresponden a una interfaz de usuario simple realizada con HTML, CSS Y JAVASCRIPT, la cual se conecta a la API REST.

¿Que es?

Es una API REST desarrollada en Java, que implementa un sistema de autenticación y autorización de usuarios, incluyendo el envío de correos electrónicos para la validación de cuentas.

¿Para que sirve?

Esta API REST proporciona un sistema seguro para la gestión de usuarios, implementando procesos de autenticación y autorización mediante Spring Security. El sistema permite controlar el acceso a recursos protegidos utilizando JWT (JSON Web Tokens), garantizando que solo los usuarios autenticados puedan interactuar con la aplicación. Además, incorpora un mecanismo de verificación de identidad a través del envío automatizado de correos electrónicos, permitiendo validar las cuentas de los usuarios antes de que puedan acceder a las funcionalidades principales del sistema.

¿Que tecnologias utiliza?
